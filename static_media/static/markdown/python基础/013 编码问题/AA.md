#### ASCII

```
8位一个字节，2^7 == 126个字符，第一位为标志位
```

#### unicode

```
所有的都是4位，存储严重缩水
```

#### utf-8

```
包含ascii码，用8位表示所有ascii
中文3个字节，24位
```

#### gbk

```
8位字符
中文：16位，两个字节
```

- 各个编码之间的二进制，是不能相互识别的，会产生乱码
- 文件的储存和传送不能是：unicode，只能是`utf  gbk  ascii`
