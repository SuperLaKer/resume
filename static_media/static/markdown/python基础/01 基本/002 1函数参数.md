#### 形参和实参

```python
def sum_num(num1, num2):  # 形参有两个作用，告诉需要传递几个值，在内部当做变量使用
    result = num1 + num2
    print("%d + %d = %d" % (num1, num2, result))
    
sum_num(10, 20)  # 实参实实在在的参数
```
