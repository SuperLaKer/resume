#### 身份运算符

- `is`：比较两个 标示符 的  内存地址  是否相等，同一个对象的引用
- 判断连个  标示符  是否引用了同一个  对象（变量名、函数名、对象名）
- `is not`

#### 和 == 的区别

`==`：用来判断两个 变量的值 是否相等

`is`：引用同一个对象，如果存在值，肯定相等

```python
a = [1, 2, 3]
b = [1, 2]

id(a) != id(b)  #True
a is b  # False,地址不相同
b.append(3)
a == b  # True

```

#### 三目运算

```
a = 5 if a > 4 else 2
print(a)  # a = 5

a = 5 if a > 6 else 2
print(a)  # a = 2
```
