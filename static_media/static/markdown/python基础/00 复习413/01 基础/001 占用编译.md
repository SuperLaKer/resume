#### ASCII

- 英文字母（字符）：一个字符占用一字节byte，8位bit
- 一个汉字：占用`2byte` == `16bit`

#### Unicode

- 英文也使用两个字节，容量缩水
- Unicode通常用两个字节表示一个字符，原有的英文编码从单字节变成双字节，只需要把高字节全部填为0就可以
- 中文字符占两个字节

#### utf-8

- 汉字3个字节
- 其他，动态分配

#### 模块的编译

- 模块---编译--->字节码（二进制）---处理--->机器码
- 字节码  源 文件的时间戳  不一样 重新编译
