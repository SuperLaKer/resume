

#### 私有属性和私有方法

- 对象的--->某些属性、方法--->只希望在 对象内部使用，不希望在外部（对象.）访问
- 属性或方法名：前面加上`__` 双下划线，就行了
- 外部访问：异常，没有方法或属性

#### 内部方法调用私有.

- 对象方法内部，可以访问：私有属性  和  私有方法

#### 底层处理访问

- 在外界可以访问  私有方法  和  私有属性
- `obj._Class__age`
- `obj._Class__secret()`

#### 父类中的私有

- 子类 对象 | 方法中， 访问不到，父类的私有
- 子类--可以间接访问--父类私有--
